# Food-Delivery-Website
# 🍔 Food Delivery Website (MERN Stack)

A **full-stack Food Delivery Web Application** built using the **MERN Stack**.
This project demonstrates a complete online food ordering system with **user authentication, cart functionality, order placement, and secure online payments using Stripe**.

The application includes:

* 👨‍🍳 Customer Frontend
* 🛠 Admin Dashboard
* ⚡ Backend API

Built using modern technologies like **React, Node.js, Express.js, MongoDB, and Stripe**.

---

# 🚀 Tech Stack

### Frontend

* React.js
* Vite
* CSS

### Backend

* Node.js
* Express.js
* MongoDB
* JWT Authentication

### Payment

* Stripe Payment Gateway

# ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/Prashant88123/Food-Delivery-Website.git
cd Food-Delivery-Website
```

---

# ▶ Running the Project

You need **three terminals** running simultaneously.

---

## 🖥 Terminal 1 — Frontend

Navigate to the frontend folder:

```bash
cd Frontend
npm install
npm run dev
```

Example terminal:

```
VITE v4.0  ready in 300ms

➜  Local:   http://localhost:5173/
```

# 🔑 Environment Variables

Create a `.env` file inside the **Backend** folder.

Example:

```
JWT_SECRET=your_secret_key
STRIPE_SECRET_KEY=your_stripe_secret
```

---

# 💳 Stripe Integration

This project uses **Stripe** for secure payment processing.

You must add your **Stripe Secret Key** in the backend `.env` file.

Official website:
[https://stripe.com](https://stripe.com)
